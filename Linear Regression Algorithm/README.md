Linear Regression Projects

This repository contains three Linear Regression projects in Python, ranging from beginner to advanced levels. Each project includes the dataset used, so you can run and understand the models easily.

Projects Overview
1. Beginner Project: Simple Linear Regression with one feature.  
2. Intermediate Project: Multiple Linear Regression with multiple features.  
3. Advanced Project: Regression with feature engineering and performance evaluation.  

Datasets
All datasets used in the projects are included in the repository for easy access.
