# Logistics Projects

This repository contains two Logistics projects in Python, along with the datasets used for each project. The projects are designed to demonstrate practical applications in logistics, from basic to intermediate complexity.

## Projects Overview
1. Project 1: Basic Logistics Analysis  
   - Focuses on simple operations and optimization tasks.
   - Includes dataset and step-by-step instructions.

2. Project 2: Intermediate Logistics Optimization  
   - Implements more advanced techniques for logistics and supply chain management.
   - Dataset included for practice and analysis.
